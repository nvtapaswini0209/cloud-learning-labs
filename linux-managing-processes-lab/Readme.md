# Managing Processes Lab (Linux on AWS EC2)

## Goal
Learn how to monitor system processes and automate tasks using cron.

## AWS Service Used
Amazon EC2

## Tasks Performed
1. Connected to EC2 using SSH
2. Generated process logs using ps command
3. Monitored system performance using top
4. Automated audit tasks using cron jobs

## Commands Used

ps -aux
grep -v root
tee
top
crontab -e

## Outcome
Created automated system audit logs for CSV files.
