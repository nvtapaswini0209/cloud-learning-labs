AWS Linux Lab – Managing File Permissions

This lab demonstrates how to manage Linux file permissions and ownership within an Amazon EC2 instance.

Tasks Covered:
• Connecting to EC2 via SSH
• Changing file and folder ownership using chown
• Modifying file permissions using chmod
• Understanding symbolic and absolute permission modes
• Assigning department-level folder permissions

Environment:
Amazon EC2 (Amazon Linux)
Instance Type: t3.micro

Commands Used:
chown
chmod
ls
vi
