# AWS Linux File Editing Lab

## Overview
In this lab, I connected to an Amazon EC2 instance and practiced editing files using Linux command-line text editors.

This lab demonstrates fundamental Linux skills required for cloud and DevOps environments.

## Objectives
- Connect to an Amazon EC2 instance using SSH
- Use Vim text editor
- Create and edit files in Linux
- Save and exit files in Vim
- Use Nano text editor

## AWS Services Used
- Amazon EC2

Instance type:
t3.micro
