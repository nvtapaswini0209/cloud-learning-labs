# AWS S3 File Sharing + SNS Notification Project

## Features
- S3 bucket creation using CLI
- IAM secure access for external user
- File upload/download/delete
- SNS email notification on changes

## Tools Used
- Amazon S3
- IAM
- SNS
- AWS CLI

## Key Commands
aws s3 mb
aws s3 sync
aws s3api put-object
aws s3api delete-object

## Outcome
- Secure file sharing implemented
- Real-time email alerts configured
