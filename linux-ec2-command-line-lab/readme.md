# AWS Software Management Lab

This lab demonstrates Linux software management and AWS CLI configuration on an EC2 instance.

## Tasks Performed
- Connected to EC2 via SSH
- Updated Linux packages using yum
- Installed Apache (httpd)
- Rolled back packages using yum history
- Installed AWS CLI
- Configured AWS CLI
- Tested AWS CLI with EC2 command

## Tools Used
- AWS EC2
- Amazon Linux
- YUM package manager
- AWS CLI
